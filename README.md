# REALMCRAFT

A browser-based, Minecraft-style first-person voxel survival sandbox with classic
pixel textures and atmospheric lighting: procedurally generated terrain and biomes,
dynamic day/night cycle, weather with thunderstorms, animated water, swaying
foliage, and fully synthesized audio. Built with TypeScript, Vite, Three.js (WebGL2).
Block textures are extracted locally from the Minecraft 26.2 client for personal
use; sounds remain generated by code.

## Run it

```bash
npm install
npm run textures:install
npm run dev
```

The original Minecraft PNG files are deliberately excluded from this public
repository. The texture installer downloads the official Mojang client, verifies
its SHA-1, and extracts only the files the game uses into the ignored `public/`
asset directory. It requires `curl`, `jq`, `unzip`, and `shasum`.

Open the printed URL (default **http://localhost:5173**), pick a seed and quality,
and click **Enter World**, then click once more to lock the pointer.

Production build + preview:

```bash
npm run build
npm run preview
```

For a lightweight runtime profile in development, open
`http://localhost:5173/?perf=1`. Every two seconds the console reports display
FPS, average CPU frame time, draw calls, triangles, loaded chunks, and movement.

## Controls

| Input | Action |
| --- | --- |
| `W A S D` | Move |
| Mouse | Look |
| `Space` | Jump / swim up / fly up |
| `Shift` | Sprint |
| `C` or `Ctrl` | Crouch / fly down |
| Left click (hold) | Break block |
| Right click | Place block |
| Mouse wheel / `1`–`9` | Select hotbar block |
| `F` | Flashlight |
| `G` | Toggle creative flight |
| `T` | Toggle time fast-forward (watch a sunset!) |
| `Y` | Cycle weather (clear → cloudy → rain → storm) |
| `Esc` | Pause / release mouse |

## Features

- **World**: seeded, chunk-streamed voxel terrain — plains, forests, deserts,
  mountains, snowfields, beaches, rivers, oceans, caves, and two tree species.
- **Graphics**: original 16×16 Minecraft block textures with nearest filtering,
  per-vertex ambient occlusion, soft dynamic shadows, ACES tone mapping,
  atmospheric sky with sun/moon/stars, drifting procedural clouds, exponential
  distance fog, animated Fresnel water with sun glints, underwater grading,
  wind-swaying grass and leaves, rain/snow particles, lightning, fireflies, birds.
- **Gameplay**: full first-person controller (sprint, crouch, swim, fly), hold-to-break
  blocks with crack stages and debris, block placement, nine-slot hotbar,
  footsteps per surface type, splash/ambience/thunder audio — all synthesized
  with the Web Audio API.
- **Performance**: face-culled greedy chunk meshes, frustum culling, budgeted
  async generation, distance-based load/unload, quality presets (Low → Ultra),
  and automatic render-distance backoff if the frame rate tanks.

## Project layout

```
src/
  core/      Game orchestrator, settings & quality presets
  world/     Blocks, chunks, procedural generation, meshing, world state
  player/    First-person controller, block interaction
  gfx/       Texture atlas, materials/shaders, sky, environment, particles
  weather/   Weather state machine
  audio/     Synthesized audio engine
  ui/        HUD, menus, screens
  util/      Math helpers, seeded simplex noise
```
