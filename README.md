# REALMCRAFT

Realmcraft — любительский браузерный voxel-sandbox в стиле Minecraft Java Edition
1.2.4. Проект написан на TypeScript и Three.js, собирается через Vite и работает
непосредственно в браузере через WebGL.

В игре уже есть процедурный бесконечный мир, биомы и пещеры, руды, режимы
Survival и Creative, здоровье и голод, инвентарь, крафт 2×2 и верстак 3×3,
печь с переплавкой и топливом, сундуки (включая двойные), фермерство и еда,
пассивные животные с AI, разведением и животными ресурсами, ближний и дальний
бой, четыре комплекта брони и шесть видов враждебных мобов,
XP-сферы, уровни и зачарования экипировки,
инструменты с прочностью и уровнями добычи, voxel-освещение, физические дропы,
сохранение мира, погода, смена времени суток и классический интерфейс Minecraft
1.2.4.

> Это не официальный продукт Mojang/Microsoft. Оригинальные ассеты Minecraft
> предназначены только для локального личного использования и не должны
> распространяться вместе с публичной сборкой проекта.

## Быстрый запуск

Требуются Node.js и npm. В текущей рабочей копии классические ассеты уже лежат в
`public/assets/minecraft`.

```bash
npm install
npm run dev
```

Vite выведет локальный адрес, обычно `http://localhost:5173`. На стартовом экране
нужно выбрать seed, качество графики и режим игры, нажать **Enter World**, а затем
щёлкнуть по игровому окну для захвата курсора.

Production-сборка и её локальный просмотр:

```bash
npm run build
npm run preview
```

### Команды npm

| Команда | Назначение |
| --- | --- |
| `npm run dev` | Запустить Vite в режиме разработки |
| `npm test` | Запустить автоматические проверки игровых систем |
| `npm run build` | Проверить TypeScript и собрать production-версию в `dist/` |
| `npm run preview` | Локально открыть production-сборку |
| `npm run textures:install` | Старый вспомогательный скрипт для современных текстур 26.2; текущий классический atlas 1.2.4 его не использует |

| Клавиша | Действие |
| --- | --- |
| `W A S D` | Передвижение |
| Мышь | Поворот камеры |
| `Space` | Прыжок, всплытие или полёт вверх |
| `Shift` | Спринт |
| `C` или `Ctrl` | Присесть или лететь вниз |

| `E` | Открыть или закрыть инвентарь в Survival |

## Режимы игры

### Survival

- 20 HP, 20 единиц голода, скрытый запас насыщения и отдельный запас воздуха под
  водой.
- Урон от падения, утопления и полного истощения голода.
- При голоде не меньше 18 здоровье постепенно восстанавливается, увеличивая
  истощение игрока.
- Спринт, плавание, прыжки, разрушение и установка блоков расходуют энергию.
- После смерти инвентарь и броня выпадают в точке гибели, опыт сбрасывается,
  а игрок возрождается в исходной точке мира с восстановленными здоровьем и голодом.
- Разрушенные блоки выпадают физическими предметами и подбираются при приближении;
  близкие одинаковые дропы объединяются, число предметов-сущностей ограничено.
- Инвентарь содержит 36 слотов, первые 9 являются хотбаром; размер стака — 64,
  инструменты не стакаются.
- Установка блока расходует один предмет из выбранного стака.

#### Еда и фермерство

- Высокая трава иногда даёт семена. ПКМ мотыгой по верхней стороне травы или
  земли создаёт грядку, а ПКМ семенами по грядке сажает пшеницу.
- Грядка увлажняется от воды в радиусе четырёх блоков. Пшеница проходит восемь
  сохраняемых стадий роста и требует уровень света не ниже 9; bone meal ускоряет
  рост.
- Созревшая пшеница даёт пшеницу и несколько семян. Из трёх единиц пшеницы
  крафтится хлеб; также доступны яблоки и грибное рагу, после которого остаётся
  миска.
- Сахарный тростник естественно встречается на мокрых берегах и вырастает до
  трёх блоков. Красные и коричневые грибы редко появляются на полу пещер.
- Листья иногда дают соответствующие саженцы, дубовые листья — ещё и яблоки.
  Саженцы вырастают по scheduled ticks; bone meal пытается вырастить дерево сразу.
- Еда употребляется удержанием ПКМ и восстанавливает голод и насыщение. Действует
  одна фиксированная модель сложности без отдельного выбора в меню: starvation
  не опускает здоровье ниже 1 HP. Сырая курица и rotten flesh могут вызвать
  временный голод, а spider eye — краткое отравление без системы зелий.

#### Животные

- В подходящих освещённых биомах группами появляются свиньи, коровы, овцы и
  куры. Общий passive mob cap и локальный лимит плотности не дают новым группам
  скапливаться в одной точке; далёкие сущности деактивируются по чанкам, а
  непостоянные могут despawn.
- Блочные модели используют оригинальные классические текстуры из
  `public/assets/minecraft/mob`, включая отдельный слой шерсти овцы.
- Животные блуждают, обходят простые препятствия, плавают, паникуют после удара и
  следуют за игроком с подходящей едой: пшеницей для свиней, коров и овец,
  семенами для кур.
- ПКМ подходящей едой переводит взрослое животное в режим размножения. Два
  животных одного вида создают детёныша, который растёт по игровым тикам;
  родители получают cooldown.
- Свиньи дают сырую свинину, коровы — говядину и кожу, овцы — баранину и шерсть,
  куры — курятину и перья. Взрослые куры периодически откладывают яйца.
- Два железных слитка крафтятся в ножницы: ими можно снять с взрослой овцы
  1–3 блока шерсти; состояние и таймер отрастания сохраняются вместе с мобом.
- Сырое мясо можно съесть сразу или приготовить в печи. Положение, здоровье,
  возраст, cooldown и стабильные id животных сохраняются вместе с миром.

#### Бой и экипировка

- При реальном уровне света не выше 7 появляются зомби, скелеты, пауки, криперы,
  слизни и Endermen: ночью на поверхности, а в тёмных пещерах — в любое время суток.
  Учитываются биом, расстояние до игрока и отдельный hostile mob cap; состояние
  существ сохраняется вместе с миром.
- Мечи увеличивают урон ближней атаки и изнашиваются. Удар во время падения
  становится критическим, а спринт усиливает knockback.
- Получивший урон моб краснеет и подскакивает; при смерти он сначала падает
  набок, и только после короткой анимации оставляет предметы и XP.
- Лук натягивается удержанием ПКМ и расходует стрелы; общая projectile-система
  также используется скелетами. Стрелы летят с гравитацией, поражают цель и
  застревают в блоках.
- В личном инвентаре есть слоты головы, груди, ног и ботинок. Кожаная, железная,
  золотая и алмазная броня имеет собственную прочность и заполняет классическую
  armor bar; полный комплект алмазной брони даёт 20 очков защиты.
- Зомби преследуют и бьют игрока, скелеты удерживают дистанцию, пауки быстрее
  атакуют в темноте, криперы взрываются, Endermen становятся агрессивными от
  взгляда или удара. Зомби и скелеты горят под открытым дневным небом.
- Боевые дропы включают rotten flesh, кости, стрелы, нити, spider eyes,
  gunpowder, slimeballs и Ender Pearls. Flint из гравия, палки и перья позволяют
  крафтить стрелы; доступны рецепты лука и всех четырёх комплектов брони.

#### Жидкости, огонь и TNT

- Вода имеет семь уровней течения, лава в обычном мире растекается в стороны до
  трёх блоков и обновляется медленнее. Два соседних источника воды создают новый
  источник над твёрдым блоком; потоки затухают после удаления источника.
- Пустое ведро набирает только источник воды или лавы; наполненное ведро размещает
  новый источник. Вёдра крафтятся из трёх железных слитков.
- Контакт воды с лавой создаёт булыжник, камень или обсидиан в зависимости от
  уровней жидкостей. Глубокие лавовые источники встречаются в пещерах ниже Y=11.
- Flint and steel поджигает соседнюю ячейку и активирует TNT. Огонь распространяется
  по горючим блокам, гаснет сам или от воды и динамически освещает окружение.
- TNT имеет сохраняемый fuse, запускает цепные реакции и использует общий движок
  взрывов: урон и knockback сущностям, разрушение блоков и физические дропы.

#### Опыт и зачарования

- Руды, существа и результат печи дают опыт. XP-сферы притягиваются к игроку,
  заполняют полосу HUD и повышают уровень; опыт и уровень сохраняются вместе с миром.
- После смерти накопленный опыт сбрасывается, а до 100 единиц остаётся на месте
  смерти в виде подбираемых сфер.
- Из сахарного тростника крафтятся бумага и книги; из книг и досок — книжные
  полки, а из книги, алмазов и обсидиана — стол зачарований.
- Книжные полки с чистым воздушным промежутком повышают силу стола до 30.
  Три детерминированных предложения стоят до 50 уровней в стиле версии 1.2.4.
- Зачарования влияют на скорость и дроп добычи, прочность, урон меча, защиту
  брони и силу лука. Данные зачарований сохраняются во всех слотах и дропах;
  зачарованные предметы имеют glint и подписи с уровнями эффектов.

#### Полный Overworld и навигация

- В Overworld есть taiga, swamp, jungle и mushroom island со своей поверхностью,
  растительностью, деревьями, огромными грибами и правилами появления существ.
- Детерминированно генерируются деревни, заброшенные шахты, подземелья и три
  stronghold с библиотеками, хранилищами, portal room и monster spawner.
- Сундуки структур используют позиционно-детерминированные loot tables. Факт их
  инициализации сохраняется отдельно, поэтому опустошённый или разрушенный сундук
  не наполняется заново после перезагрузки чанка.
- В деревнях живут сохраняемые villagers пяти базовых профессий; они гуляют около
  дома и убегают после нападения. На mushroom island появляются mooshroom, а
  враждебный естественный спавн там отключён.
- Кровать ставится двумя половинами. Сон доступен ночью при отсутствии монстров
  рядом, переводит время к рассвету и задаёт личную точку respawn.
- Компас показывает направление и расстояние до кровати или мирового spawn,
  часы — игровое время, карта — пиксельный срез местности вокруг игрока.

#### Крафт и инструменты

- В инвентаре доступна личная сетка крафта 2×2; верстак открывает сетку 3×3
  (ПКМ по установленному верстаку; с присяданием блок ставится поверх него —
  то же правило действует для печи и сундука).
- ЛКМ по слоту берёт или кладёт весь стек, ПКМ берёт половину или кладёт один
  предмет; Shift-click быстро переносит стек между инвентарём, бронёй и контейнером,
  а клик за пределами окна выбрасывает стек или один предмет. При закрытии окна
  содержимое сетки возвращается в инвентарь.
- Рецепты: доски из брёвен, палки, верстак, печь, сундук, факелы, миски, хлеб,
  грибное рагу, сахар, bone meal, лук, стрелы, четыре комплекта брони и полный набор
  инструментов — кирки, топоры, лопаты, мотыги и мечи из дерева, камня, железа,
  золота и алмаза.
- В верстаке есть книга рецептов (кнопка-книга в правом верхнем углу окна):
  панель показывает все рецепты, недоступные по материалам затемняются,
  клик по доступному автоматически раскладывает ингредиенты из инвентаря по сетке. Готовой
  текстуры recipe book в ассетах 1.2.4 нет (механика появилась в 1.12), поэтому
  панель собрана из классической палитры GUI и иконки книги из `gui/items.png`.
- Инструменты ускоряют добычу подходящих блоков, имеют прочность и ломаются;
  износ отображается полоской в слоте и сохраняется в мире.
- Уровни добычи: каменные блоки требуют кирку, железная руда — каменную кирку,
  золотая и алмазная руды — железную. Без подходящего инструмента блок ломается
  медленно и не даёт дроп.
- Правильные дропы: камень → булыжник, трава → земля, угольная руда → уголь,
  алмазная руда → алмаз, листья и стекло не выпадают; железная и золотая руды
  падают блоками и переплавляются в печи.

#### Печь и сундуки

- ПКМ по печи открывает классическое окно с тремя слотами (сырьё, топливо,
  результат), индикатором пламени и стрелкой прогресса; каждый предмет
  переплавляется 10 секунд.
- Рецепты переплавки: железная и золотая руда → слитки, песок → стекло,
  булыжник → камень; сырая свинина, говядина, курятина и баранина →
  соответствующее приготовленное мясо.
- Топливо горит разное время: уголь 80 с, доски и брёвна 15 с, деревянные
  инструменты 10 с, палки 5 с.
- Печь продолжает работать при закрытом окне и после перезагрузки страницы;
  поставленная печь разворачивается к игроку, а горящая передняя грань светится и
  слегка мерцает.
- Сундук хранит 27 слотов. Два сундука, поставленные вплотную, образуют двойной
  сундук на 54 слота с единой большой моделью; сундуки разворачиваются к игроку,
  а рядом с двойным сундуком новый сундук поставить нельзя.
- При разрушении печи или сундука всё содержимое выпадает предметами.

Цветы, высокая трава, факелы и другие cross-элементы отображаются плоскими
прозрачными спрайтами в инвентаре, руке и среди выпавших предметов. Тонкая
контурная рамка показывает выбранный блок, а при удержании ЛКМ поверх него
последовательно отображаются 10 стадий разрушения.

### Creative

- Урон, голод и конечные стаки отключены.
- Доступен бесконечный набор блоков на нескольких страницах хотбара.
- `R` переключает страницы, `G` включает обычный Creative-полёт.

### Xray / noclip

`X` включает служебный режим осмотра генерации руд. Обычные блоки становятся
прозрачными, руды остаются видимыми, столкновения и гравитация отключаются. Режим
доступен для тестирования и в Creative, и в Survival; в Survival во время noclip
не расходуется голод и не применяется урон от окружения.

## HUD и интерфейс

Интерфейс повторяет логические размеры и pixel-art стиль Minecraft 1.2.4:

- bitmap-шрифт загружается из `font/default.png`;
- масштаб GUI выбирается целым множителем с учётом размера окна;
- хотбар и рамка выбранного слота используют `gui/gui.png`;
- сердца, броня, голод, воздух и полоса опыта используют `gui/icons.png`;
- окно Survival-инвентаря использует `gui/inventory.png`;
- иконки блоков строятся из тех же atlas-тайлов, что и блоки в мире;
- при получении урона появляются красная вспышка, покачивание камеры, анимация
  сердец и классический звук боли.

Полоса опыта показывает текущий прогресс и уровень; XP-сферы используют
классическую текстуру `item/xporb.png`, а стол — окно `gui/enchant.png`.

В левом верхнем углу выводятся FPS, координаты, биом, игровое время, погода и seed.
Все строки используют один размер bitmap-шрифта.

## Мир и генерация

Мир детерминирован seed-строкой и генерируется чанками. Один и тот же seed создаёт
тот же базовый ландшафт; пользовательские изменения сохраняются отдельно.

Реализованы:

- ocean, beach, plains, forest, desert, mountains, snowfields и river;
- холмы, берега, океаны, реки и системы пещер с редкими естественными входами
  на суше;
- дубы и ели, высокая трава и цветы;
- сахарный тростник на мокрых берегах и редкие грибы в пещерах;
- вода с уровнями течения, плавание и подводный визуальный эффект;
- глубокие источники лавы, динамический свет и образование камня, булыжника и
  обсидиана при контакте жидкостей;
- падающие песок и гравий;
- угольная, железная, золотая и алмазная руда.

Мир обновляется фиксированными тактами 20 Гц. Для активных чанков работают
random ticks, очередь обязательных block ticks и уведомления соседних блоков.
Skylight и свет факелов распространяются между voxel-ячейками в диапазоне 0–15;
единый запрос освещения используется ростом растений и затенением геометрии.

### Высоты и частота руд

| Руда | Высота Y | Попыток жил на чанк | Размер жилы |
| --- | ---: | ---: | ---: |
| Уголь | 5–96 | 10 | 8–16 |
| Железо | 5–64 | 8 | 5–10 |
| Золото | 5–32 | 2 | 4–7 |
| Алмазы | 5–18 | 1 | 3–6 |

Жилы генерируются детерминированно после пещер и заменяют только камень, поэтому
часть руды естественным образом видна на стенах пещер.

## Сохранения

Состояние хранится в браузерном `localStorage`, отдельно для каждого seed.

- Ключ мира: `realmcraft.world.v1.<encoded-seed>`.
- Ключ настроек: `realmcraft.settings.v1`.
- Автосохранение выполняется каждые 8 секунд, при паузе, скрытии/закрытии страницы
  и выходе в главное меню.
- Сохраняются позиция и направление игрока, HP, голод, насыщение, воздух, опыт,
  инвентарь, надетая броня, лежащие
  в мире предметы, содержимое сундуков и печей вместе с прогрессом переплавки,
  выбранный слот/страница, время суток, погода, noclip и все изменённые игроком
  блоки и зачарования предметов, включая направление печей и сундуков, возраст культур, влажность почвы,
  очередь scheduled block ticks, а также все существа с их здоровьем, возрастом,
  cooldown размножения и боевым состоянием.
- Качество, громкость, head bob, последний seed и последний выбранный режим
  сохраняются как настройки.

Очистка данных сайта в браузере удалит сохранённые миры.

## Классические ассеты Minecraft 1.2.4

Текущий renderer использует старый формат ресурсов до Minecraft 1.5.

### Текстуры

Главный источник блоков — `public/assets/minecraft/terrain.png`. Во время запуска
`src/gfx/Atlas.ts` выбирает из него необходимые тайлы и собирает компактный runtime
atlas 8×8 с размером тайла 16×16. Там же создаются стадии разрушения блоков и
цветовые варианты травы/листвы.

Файлы, которые непосредственно загружает текущий код:

```text
public/assets/minecraft/
  terrain.png
  font/default.png
  gui/gui.png
  gui/icons.png
  gui/inventory.png
  gui/crafting.png
  gui/furnace.png
  gui/container.png
  gui/enchant.png
  gui/items.png
  sound/...
```

Остальные каталоги старого клиента (`mob`, `armor`, `art`, `title`, `item` и
другие) сохранены как исходный набор для будущих этапов, но сейчас напрямую не
используются.

### Звуки

В локальном старом client JAR звуки не находились: лаунчер Minecraft загружал их
отдельно в папку `%APPDATA%\.minecraft\resources`. Поэтому звуки были восстановлены
через официальный Mojang asset index `pre-1.6`:

- [официальный индекс pre-1.6](https://launchermeta.mojang.com/v1/packages/3d8e55480977e32acd9844e545177e69a52f594b/pre-1.6.json);
- файлы загружаются с `https://resources.download.minecraft.net/<2 символа SHA-1>/<SHA-1>`;
- локальная папка проекта: `public/assets/minecraft/sound`.

В проекте находятся 134 оригинальных OGG-файла:

```text
sound/
  step/               cloth, grass, gravel, sand, snow, stone, wood (по 4)
  damage/             fallbig, fallsmall, hurtflesh
  liquid/             splash
  random/             bow, click, pop, eat, explode, fuse, chestopen, orb
  ambient/weather/    rain, thunder
  mob/                 животные, zombie, skeleton, spider, creeper, slime, enderman
  music/               calm1, calm2, calm3
```

`src/audio/Audio.ts` асинхронно декодирует их через Web Audio API. Оригинальные
семплы используются для шагов, блоков, падения, урона, воды, предметов, погоды,
еды, лука, сундуков, опыта, взрывов, мобов и фоновой музыки. Если конкретный файл
не загрузился, соответствующее событие остаётся без звука — синтезированных
звуковых подмен нет. Ветер и птицы являются только визуальной частью окружения.

## Графика и погода

- Three.js/WebGL, nearest filtering и pixel-art текстуры.
- Greedy meshing, отсечение невидимых граней и frustum culling.
- Ambient occlusion на вершинах и динамические тени.
- ACES tone mapping, дистанционный туман и атмосферное небо.
- Солнце, луна, звёзды, процедурные облака и смена дня/ночи.
- Полный цикл суток длится 20 реальных минут при обычной скорости; `T` оставлен
  как служебное ускорение времени.
- Анимированная вода с отражённым бликом и подводной цветокоррекцией.
- Ветер для травы/листвы, дождь, снег, молнии, светлячки и декоративные птицы.

### Профили качества

| Профиль | Дальность, чанки | Shadow map | Pixel ratio cap | Плотность травы |
| --- | ---: | ---: | ---: | ---: |
| Low | 4 | 1024 | 1.0 | 35% |
| Medium | 6 | 2048 | 1.25 | 60% |
| High | 7 | 2048 | 1.5 | 100% |
| Ultra | 10 | 4096 | 2.0 | 100% |

Если FPS остаётся ниже 25 больше восьми секунд, игра автоматически уменьшает
дальность прорисовки.

Для вывода метрик производительности можно открыть:

```text
http://localhost:5173/?perf=1
```

Раз в две секунды в консоль выводятся FPS, CPU frame time, draw calls, triangles,
число загруженных чанков и дополнительные runtime-метрики.

## Текущие ограничения

- Нет зелий, измерений Nether/End и крупных структур.
- Survival пока является фундаментом для следующих игровых систем, а не полной
  реализацией оригинального Minecraft.
